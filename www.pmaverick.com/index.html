Project Maverick

Project Maverick 
Truly Omni-Directional

Project Maverick is an omni-directional robotic system designed to provide a mobility solution for people with walking disabilities by mimicking the movement patterns of humans. The drive system allows the user to move in any direction using 4 steering and 4 driving electronically synchronized motors, creating the same degrees of motion as an able person.
